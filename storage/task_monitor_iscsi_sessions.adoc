= Ver sesiones iSCSI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puede ver las sesiones iSCSI que están conectadas al clúster.  Puedes filtrar la información para incluir solo las sesiones deseadas.

. En la interfaz de usuario de Element, seleccione *Informes* > *Sesiones iSCSI*.
. Para ver los campos de criterios de filtro, haga clic en *Filtrar*.
