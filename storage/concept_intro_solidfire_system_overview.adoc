= Sistema de almacenamiento SolidFire
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Un sistema de almacenamiento all-flash SolidFire se compone de componentes de hardware discretos (unidades y nodos) que se combinan en un único grupo de recursos de almacenamiento a través del software NetApp Element, que se ejecuta de forma independiente en cada nodo.  Este clúster unificado se presenta como un único sistema de almacenamiento para uso de clientes externos y se gestiona como una sola entidad a través de la interfaz de usuario del software Element, la API y otras herramientas de gestión.

Mediante la interfaz de usuario del software NetApp Element , puede configurar y supervisar la capacidad y el rendimiento del almacenamiento del clúster SolidFire , y gestionar la actividad de almacenamiento en una infraestructura multiinquilino.

image::../media/solidfire_concepts_architecture_image.gif[Diagrama arquitectónico de Element Software]

Un sistema de almacenamiento SolidFire totalmente flash incluye los siguientes componentes:

* Nodos: Hardware físico que proporciona los recursos de almacenamiento para un clúster.  Existen dos tipos de nodos:
+
** Nodos de almacenamiento: Servidores que contienen una colección de unidades.
** Nodos Fibre Channel (FC): Se utilizan para conectar clientes FC a través de un conmutador Fibre Channel.


* Clúster: El núcleo del sistema de almacenamiento SolidFire , compuesto por al menos cuatro nodos.
* Nodo de administración: Le permite actualizar y proporcionar servicios del sistema, incluyendo monitoreo y telemetría, administrar los activos y la configuración del clúster, ejecutar pruebas y utilidades del sistema y proporcionar acceso al soporte de NetApp para la resolución de problemas.  El nodo de gestión (mNode) es una máquina virtual que se ejecuta en conjunto con un clúster de almacenamiento basado en software Element.
* Active IQ: Una herramienta basada en la web que proporciona vistas históricas actualizadas continuamente de los datos de todo el clúster.  Puedes configurar alertas para eventos, umbrales o métricas específicas.  Active IQ le permite supervisar el rendimiento y la capacidad del sistema, así como mantenerse informado sobre el estado del clúster.
* Las unidades se utilizan en los nodos de almacenamiento y almacenan datos para el clúster.  Un nodo de almacenamiento contiene dos tipos de unidades:
+
** Las unidades de metadatos de volumen almacenan información que define los volúmenes y otros objetos dentro de un clúster.
** Las unidades de bloque almacenan bloques de datos para volúmenes de aplicaciones.
