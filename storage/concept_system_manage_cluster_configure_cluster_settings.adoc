= Configurar los ajustes del clúster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Desde la pestaña Clúster de la interfaz de usuario de Element, puede ver y cambiar la configuración de todo el clúster y realizar tareas específicas del clúster.

Puede configurar ajustes como el umbral de llenado del clúster, el acceso de soporte, el cifrado en reposo, los volúmenes virtuales, SnapMirror y el cliente de difusión NTP.
