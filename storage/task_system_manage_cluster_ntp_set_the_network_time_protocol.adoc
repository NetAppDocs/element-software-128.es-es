= Configurar el protocolo de tiempo de red
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La configuración del Protocolo de Tiempo de Red (NTP) se puede lograr de dos maneras: o bien instruyendo a cada nodo de un clúster para que escuche las transmisiones o bien instruyendo a cada nodo para que consulte a un servidor NTP para obtener actualizaciones.

El protocolo NTP se utiliza para sincronizar los relojes a través de una red.  La conexión a un servidor NTP interno o externo debe formar parte de la configuración inicial del clúster.
