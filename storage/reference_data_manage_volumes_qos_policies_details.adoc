= Detalles de las políticas de QoS
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes ver los detalles de las políticas de QoS desde la pestaña Administración.

* *IDENTIFICACIÓN*
+
El ID generado por el sistema para la política de QoS.

* *Nombre*
+
El nombre definido por el usuario para la política de QoS.

* *IOPS mínimas*
+
El número mínimo de IOPS garantizado para el volumen.

* *Máximo de IOPS*
+
El número máximo de IOPS permitido para el volumen.

* *Ráfaga de IOPS*
+
El número máximo de IOPS permitido durante un corto período de tiempo para el volumen.  Valor predeterminado = 15.000.

* *Volúmenes*
+
Muestra el número de volúmenes que utilizan la política.  Este número enlaza con una tabla de volúmenes a los que se les ha aplicado la política.
