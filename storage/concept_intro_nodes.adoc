= Nodos
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Los nodos son los componentes de hardware individuales que se agrupan en un clúster para ser accedidos como almacenamiento en bloque.  En un sistema de almacenamiento SolidFire existen dos tipos de nodos: nodos de almacenamiento y nodos Fibre Channel.
