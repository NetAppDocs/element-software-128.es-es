= Restablecer cifrados TLS suplementarios
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `ResetSupplementalTlsCiphers` Método para restaurar la lista de cifrados TLS suplementarios a la predeterminada.  Puedes utilizar este método en todo el clúster.



== Parámetro

Este método no tiene parámetros de entrada.



== Valores de retorno

Este método no devuelve ningún valor.



== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
  "method": "ResetSupplementalTlsCiphers",
  "params": {},
  "id" : 1
}
----


== Ejemplo de respuesta

Este método devuelve una respuesta similar al siguiente ejemplo:

[listing]
----
{
  "id" : 1,
  "result" : {}
}
----

11,3
