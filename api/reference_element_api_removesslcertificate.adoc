= Eliminar certificado SSL
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `RemoveSSLCertificate` Método para eliminar el certificado SSL de usuario y la clave privada para los nodos de almacenamiento en el clúster.  Una vez eliminados el certificado y la clave privada, los nodos de almacenamiento se configuran para usar el certificado y la clave privada predeterminados.



== Parámetros

Este método no tiene parámetros de entrada.



== Valores de retorno

Este método no devuelve ningún valor.



== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
    "method" : "RemoveSSLCertificate",
    "params" : {},
    "id" : 3
}
----


== Ejemplo de respuesta

Este método devuelve una respuesta similar al siguiente ejemplo:

[listing]
----
{
    "id" : 3,
    "result" : {}
}
----
