= Regla de política de espejo snap
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
El objeto snapMirrorPolicyRule contiene información sobre las reglas de una política de SnapMirror .



== miembros del objeto

Este objeto contiene los siguientes miembros:

|===
| Nombre | Descripción | Tipo 


 a| 
snapMirrorLabel
 a| 
La etiqueta de copia de instantánea, utilizada para la selección de copias de instantáneas en relaciones de protección de datos extendidas.
 a| 
cadena



 a| 
keepCount
 a| 
Especifica el número máximo de copias de instantáneas que se conservan en el volumen de destino de SnapMirror para una regla.
 a| 
entero

|===
