= Lista todos los nodos
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `ListAllNodes` Método para listar los nodos activos y pendientes en el clúster.



== Parámetros

Este método no tiene parámetros de entrada.



== Valores de retorno

Este método tiene los siguientes valores de retorno:

|===
| Nombre | Descripción | Tipo 


 a| 
nodos
 a| 
Lista de objetos que describen los nodos activos en el clúster.
 a| 
xref:reference_element_api_node.adoc[nodo]



 a| 
nodos activos pendientes
 a| 
Lista de objetos que describen los nodos activos pendientes para el clúster.
 a| 
xref:reference_element_api_pendingactivenode.adoc[nodo activo pendiente]formación



 a| 
nodos pendientes
 a| 
Lista de objetos que describen los nodos pendientes para el clúster.
 a| 
xref:reference_element_api_pendingnode.adoc[nodo pendiente]formación

|===


== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
   "method": "ListAllNodes",
   "params": {},
   "id" : 1
}
----
