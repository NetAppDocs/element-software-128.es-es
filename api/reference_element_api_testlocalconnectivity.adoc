= Prueba de conectividad local
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `TestLocalConnectivity` método para hacer ping a la IP del clúster (CIP) de cada nodo en un clúster activo.



== Parámetros

Este método no tiene parámetros de entrada.



== Valor de retorno

Este método tiene el siguiente valor de retorno:

|===
| Nombre | Descripción | Tipo 


 a| 
detalles
 a| 
Tiempos de respuesta individuales al ping para cada nodo en el clúster local activo.
 a| 
Objeto JSON

|===


== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
    "method": "TestLocalConnectivity",
    "params": {},
    "id": 1
}
----
