= Métodos de la API de SnapMirror
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Los métodos de la API SnapMirror son utilizados por la interfaz web de Element para administrar las instantáneas replicadas con sistemas ONTAP remotos.  Estos métodos están destinados únicamente para su uso por la interfaz web de Element.  Si necesita acceso a la API de la funcionalidad de SnapMirror , utilice las API de ONTAP .
