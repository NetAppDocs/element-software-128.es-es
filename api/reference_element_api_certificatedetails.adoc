= Detalles del certificado
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
El `certificateDetails` El objeto contiene la información decodificada sobre un certificado de seguridad.



== miembros del objeto

Este objeto contiene los siguientes miembros:

|===
| Nombre | Descripción | Tipo 


 a| 
`issuer`
 a| 
El nombre del emisor.
 a| 
cadena



 a| 
`modulus`
 a| 
El módulo de la clave pública.
 a| 
cadena



 a| 
`notAfter`
 a| 
La fecha de vencimiento del certificado.
 a| 
Cadena ISO 8601



 a| 
`notBefore`
 a| 
La fecha de inicio del certificado.
 a| 
Cadena ISO 8601



 a| 
`serial`
 a| 
El número de serie del certificado.
 a| 
cadena



 a| 
`sha1Fingerprint`
 a| 
Resumen de la versión del certificado codificada en DER.
 a| 
cadena



 a| 
`subject`
 a| 
El nombre del sujeto.
 a| 
cadena

|===
