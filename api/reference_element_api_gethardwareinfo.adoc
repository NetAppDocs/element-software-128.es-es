= Obtener información de hardware
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `GetHardwareInfo` Método para obtener información y estado del hardware en tiempo real para un solo nodo.  La información sobre el hardware generalmente incluye fabricantes, proveedores, versiones, unidades y otra información de identificación asociada.



== Parámetro

Este método tiene el siguiente parámetro de entrada:

|===
| Nombre | Descripción | Tipo | Valor predeterminado | Requerido 


 a| 
fuerza
 a| 
Establezca este parámetro "force" en verdadero para ejecutarlo en todos los nodos del clúster.
 a| 
booleano
 a| 
FALSO
 a| 
No

|===


== Valor de retorno

Este método tiene el siguiente valor de retorno:

|===
| Nombre | Descripción | Tipo 


 a| 
Información de hardware
 a| 
Información de hardware del nodo.
 a| 
xref:reference_element_api_hardwareinfo.adoc[Información de hardware]

|===


== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
   "method": "GetHardwareInfo",
   "params": {
   },
"id" : 1
}
----
