= error
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
El objeto de error contiene un código y un mensaje de error si se produce un error durante una llamada a un método.  Todos los errores generados por el sistema tienen un código de error 500.



== miembros del objeto

Este objeto contiene los siguientes miembros:

|===
| Nombre | Descripción | Tipo 


 a| 
código
 a| 
El código numérico utilizado para identificar el error.  Todos los errores generados por el sistema devuelven un código 500.
 a| 
entero



 a| 
nombre
 a| 
El identificador único del error específico que se produjo.  Cada método devuelve un conjunto documentado de errores, aunque también debes estar preparado para manejar errores no reconocidos.
 a| 
cadena



 a| 
mensaje
 a| 
Una descripción del error, posiblemente con detalles adicionales.
 a| 
cadena

|===
