= Información sobre el hardware de la unidad
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `GetDriveHardwareInfo` Método para obtener toda la información de hardware de la unidad dada.  Esto generalmente incluye fabricantes, vendedores, versiones y otra información de identificación de hardware asociada.



== Parámetro

Este método tiene el siguiente parámetro de entrada:

|===
| Nombre | Descripción | Tipo | Valor predeterminado | Requerido 


 a| 
driveID
 a| 
Identificador de la unidad para la solicitud.
 a| 
entero
 a| 
Ninguno
 a| 
Sí

|===


== Valor de retorno

Este método tiene el siguiente valor de retorno:

|===
| Nombre | Descripción | Tipo 


 a| 
resultado
 a| 
Se devolvió la información de hardware para el driveID especificado.
 a| 
xref:reference_element_api_hardwareinfo.adoc[Información de hardware]

|===


== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
   "method": "GetDriveHardwareInfo",
   "params": {
     "driveID": 5
   },
   "id" : 100
}
----

   "result" : {

       "dev" : "8:80",

       "driveSecurityAtMaximum" : false,
       "driveSecurityFrozen" : false
       "driveSecurityLocked" : false,
       "logicalname" : "/dev/sdf",
       "product" : "INTEL SSDSA2CW300G3",

   }
}
----
