= ObtenerInterfazDeRed
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `GetNetworkInterface` Método para obtener información sobre una interfaz de red en un nodo.



== Parámetros

Este método tiene los siguientes parámetros de entrada:

|===
| Nombre | Descripción | Tipo | Valor predeterminado | Requerido 


 a| 
interfaz
 a| 
El nombre de la interfaz para obtener información sobre cada nodo. Valores posibles:

* Bond1G
* Bond10G

 a| 
cadena
 a| 
Ninguno
 a| 
No



 a| 
fuerza
 a| 
Establezca este parámetro en verdadero para ejecutarlo en todos los nodos del clúster.
 a| 
booleano
 a| 
FALSO
 a| 
No

|===


== Valor de retorno

Este método tiene el siguiente valor de retorno:

|===
| Nombre | Descripción | Tipo 


 a| 
nodos
 a| 
Una matriz de objetos que describe la interfaz de cada nodo de almacenamiento en el clúster de almacenamiento.  Cada objeto dentro de la matriz contiene los siguientes elementos:

* nodeID: (entero) El ID del nodo de almacenamiento en el clúster de almacenamiento al que se aplica la información de la interfaz.
* resultado: (xref:reference_element_api_networkinterface.adoc[Interfaz de red] ) Información de configuración de la interfaz para este nodo de almacenamiento.

 a| 
matriz de objetos JSON

|===


== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
	"method": "GetNetworkInterface",
	"params": {
		"interface": "Bond1G",
		"force": true
		},
	"id": 1
}
----
