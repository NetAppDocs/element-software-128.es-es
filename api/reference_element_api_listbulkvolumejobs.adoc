= Lista de trabajos de volumen masivo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puedes usar el `ListBulkVolumeJobs` Método para obtener información sobre cada operación de lectura o escritura de volumen masivo que se produce en el sistema.



== Parámetros

Este método no tiene parámetros de entrada.



== Valor de retorno

Este método tiene el siguiente valor de retorno:

|===
| Nombre | Descripción | Tipo 


 a| 
Trabajos de gran volumen
 a| 
Un conjunto de información para cada trabajo de gran volumen.
 a| 
xref:reference_element_api_bulkvolumejob.adoc[Trabajo de gran volumen]formación

|===


== Ejemplo de solicitud

Las solicitudes para este método son similares al siguiente ejemplo:

[listing]
----
{
   "method": "ListBulkVolumeJobs",
   "params": {
      },
   "id" : 1
}
----
