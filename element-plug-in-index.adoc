= Plugin de NetApp Element para vCenter Server
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
El complemento NetApp Element para vCenter Server proporciona un complemento a la interfaz de VMware vSphere para que pueda administrar y supervisar clústeres de almacenamiento que ejecutan el software NetApp Element .

Para obtener información sobre el complemento Element para vCenter Server, consulte la https://docs.netapp.com/us-en/vcp/index.html["Documentación del complemento NetApp Element para vCenter Server"^] .
